# Facebook clone

Facebook UI clone using Flutter.

> this repo has two branches they do the same thing and have same design but master branch is using *Cubit* for state management
> and the second branch -from the name 'with_bloc'- is using *Bloc* for state management.
> When I started i used Cubit for the sake of trying it but the original task requires using bloc so i did it in a second branch.
